# Task-Lisk
- Full Stack Project Management Webapp
- View all to-do's in easy to understand groupings
- Easily modify grouping of to-do's
- Add new to-do's
- Remove old to-do's
- Click a to-do to view addtional details

# Features
- Drag and Drop Tasks
- Light Mode & Dark Mode

# Delpoyment
- Heroku
